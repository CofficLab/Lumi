## ✨ 功能特性

Lumi 致力于成为你得力的 macOS 桌面助理，提供以下核心能力：

- **设备状态监控**：实时掌握 CPU、内存、磁盘、电池等关键硬件指标与系统信息，对设备健康状况了如指掌。
- **防休眠控制**：一键阻止系统休眠，支持灵活的模式切换与时长设置，确保重要任务不因息屏而中断。
- **状态栏助理**：在菜单栏常驻显示时间、版本信息与后台活动状态，无需切换窗口即可获取关键信息。
- **应用与项目管理**：快速查看应用元数据，便捷管理当前项目与标签页信息，提升工作流效率。
- **个性化体验**：根据个人需求在设置中灵活开启或关闭各项功能模块，打造专属的助理界面。

## 📋 系统要求

- macOS 13.0+
- Xcode 15.0+
- Swift 5.9+

## 🚀 构建与运行

1. **克隆仓库**
   ```bash
   git clone https://github.com/Coffic/Lumi.git
   cd Lumi
   ```

2. **在 Xcode 中打开**
   ```bash
   open Lumi.xcodeproj
   ```

3. **构建与运行**
   - 选择合适的 macOS 目标
   - 构建 (⌘B) 并运行 (⌘R)
