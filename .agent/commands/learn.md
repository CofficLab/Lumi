# 代码学习向导

帮助用户学习和理解代码库。

## 工作流程

### 1. 确定学习目标

询问用户想要了解：
- 特定功能
- 特定模块
- 整体架构
- 最佳实践

### 2. 探索代码

- 查看目录结构
- 阅读关键文件
- 分析代码流程

### 3. 解释说明

- 使用简单清晰的语言
- 提供代码示例
- 标注关键部分
- 解释设计决策

### 4. 总结要点

- 核心概念
- 关键代码
- 注意事项

## 规则

- 由浅入深，循序渐进
- 多使用图表和示例
- 避免过长的代码块
- 用中文解释
