# 三栏布局实现说明

## 功能概述

在 Agent 模式下实现了三栏布局：
- **左侧栏（Sidebar）**：220px，显示插件提供的侧边栏视图（如项目文件树）
- **中间栏（Middle）**：300px，显示插件提供的中间视图（如文件预览）
- **右侧栏（Detail）**：弹性宽度，显示插件提供的详情视图

## 实现细节

### 1. AgentProvider 扩展

添加了文件选择相关的状态管理：
- `selectedFileURL`: 当前选择的文件 URL
- `selectedFilePath`: 当前选择的文件路径
- `selectedFileContent`: 当前选择的文件内容
- `isFileSelected`: 是否已选择文件
- `selectFile(at:)`: 选择文件并加载内容
- `clearFileSelection()`: 清除文件选择

### 2. PluginProvider 扩展

添加了 `getMiddleViews()` 方法，用于获取插件提供的中间栏视图。

### 3. SuperPlugin 协议扩展

添加了 `addMiddleView()` 方法，允许插件提供中间栏视图。

### 4. ContentView 布局修改

在 `agentModeLayout` 中实现了三栏布局：
```swift
HStack(spacing: 0) {
    // 第一栏：侧边栏 (220px)
    if sidebarVisibility { ... }
    
    // 第二栏：中间栏 (300px)
    if !middleViews.isEmpty { ... }
    
    // 第三栏：详情栏 (弹性宽度)
    agentDetailContent()
}
```

### 5. 新增 FilePreviewPlugin 插件

创建了新的文件预览插件：
- **FilePreviewPlugin.swift**: 插件定义，提供中间栏视图
- **Views/FilePreviewView.swift**: 文件预览视图，显示选中文件的信息和内容

### 6. ProjectTreeView 更新

修改了文件树视图以支持文件选择：
- 添加了 `onFileSelect` 回调
- 点击文件时更新 `AgentProvider.selectedFileURL`
- 高亮显示当前选中的文件

## 使用方式

1. 切换到 Agent 模式
2. 在左侧文件树中点击任意文件
3. 中间栏会自动显示文件预览（文件名、路径、内容）
4. 点击中间栏的清除按钮可以取消文件选择

## 插件开发

如果其他插件想要提供中间栏视图，只需实现 `addMiddleView()` 方法：

```swift
@MainActor func addMiddleView() -> AnyView? {
    return AnyView(MyMiddleView())
}
```

## 文件结构

```
LumiApp/
├── Core/
│   ├── Providers/
│   │   ├── AgentProvider.swift (扩展了文件选择功能)
│   │   └── PluginProvider.swift (添加了 getMiddleViews)
│   ├── Contact/
│   │   └── SuperPlugin.swift (添加了 addMiddleView 协议方法)
│   └── Views/Layout/
│       └── ContentView.swift (实现三栏布局)
└── Plugins/
    ├── FilePreviewPlugin/ (新增)
    │   ├── FilePreviewPlugin.swift
    │   └── Views/
    │       └── FilePreviewView.swift
    └── ProjectTreePlugin/
        └── Views/
            └── ProjectTreeView.swift (更新支持文件选择)
```

## 注意事项

- 中间栏只在有插件提供视图时显示
- 文件内容加载是异步的，大文件可能需要一点时间
- 文件树只加载文本文件内容，二进制文件会显示错误提示
