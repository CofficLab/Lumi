# VideoConverterPlugin

Lumi plugin: Video file conversion.

## Features

- **Video Conversion** — Convert video files between formats
- **Format Support** — Various video format support

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
