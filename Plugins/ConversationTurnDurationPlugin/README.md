# ConversationTurnDurationPlugin

对话轮次时长显示插件，在 Chat 工具栏显示当前轮次已持续的时间。

## 功能特性

- 🕐 **实时显示**：在聊天工具栏中实时显示当前对话轮次的持续时间
- 🎯 **自动检测**：自动检测轮次开始（用户发送消息）和结束（AI 完成回复）
- ⏱️ **精确计时**：从最后一条用户消息的时间开始计时，每秒更新
- 🎨 **视觉反馈**：轮次进行中时高亮显示，空闲时显示灰色
- 🌍 **多语言支持**：支持英文、简体中文、繁体中文

## 显示格式

- 少于 1 分钟：`0:30`（分:秒）
- 1-59 分钟：`5:30`（分:秒）
- 超过 1 小时：`1:05:30`（时:分:秒）

## 实现原理

1. **监听机制**：通过监听 `ChatService.$revision` 的变化来检测新轮次
2. **起始时间**：使用会话中最后一条用户消息的 `createdAt` 作为轮次开始时间
3. **状态判断**：通过 `isSending(for:)` 判断轮次是否正在进行
4. **实时更新**：使用 `Timer` 每秒更新显示的时长

## 插件配置

- **ID**: `com.coffic.lumi.plugin.conversation-turn-duration`
- **分类**: Agent
- **策略**: Always On（始终启用）
- **阶段**: Beta
- **顺序**: 86（在 Chat 工具栏中的位置）

## 技术要点

- 无需修改内核代码，完全在插件层面实现
- 符合"最少功能原则"，保持代码简洁
- 性能开销小，仅在需要时启动定时器
- 线程安全，所有 UI 更新都在 MainActor 上执行
