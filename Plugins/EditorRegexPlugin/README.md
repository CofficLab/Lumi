# EditorRegexPlugin

Lumi plugin: Regular expression support for the editor.

## Features

- **Syntax Highlighting** — Regex syntax highlighting
- **Language Support** — Regular expression language integration

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [EditorService](../../Packages/EditorService) | Editor service |
