# GitHubPlugin

Lumi plugin: GitHub integration.

## Features

- **Repository Access** — Access GitHub repositories
- **Issue Management** — View and manage GitHub issues
- **PR Support** — Pull request integration

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [GitHubKit](../../Packages/GitHubKit) | GitHub API integration |
