# EditorMinimapContextMenuPlugin

Lumi plugin: Context menu for the editor minimap.

## Features

- **Context Menu** — Right-click menu on minimap
- **Quick Actions** — Quick navigation and actions via minimap

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [EditorService](../../Packages/EditorService) | Editor service |
