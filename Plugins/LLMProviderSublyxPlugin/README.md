# LLMProviderSublyxPlugin

Lumi plugin: Sublyx LLM provider integration.

## Features

- **LLM Provider** — registers Sublyx in the model selector
- **Model catalog** — provides available models for Sublyx

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [LLMKit](../../Packages/LLMKit) | LLM service abstractions |
| [LLMProviderKit](../../Packages/LLMProviderKit) | LLM provider protocol definitions |
