# ActivityHeatmapPlugin

Lumi plugin: Activity Heatmap visualization.

## Features

- **Heatmap Display** — Shows activity intensity over time
- **Visualization** — Calendar-based activity representation

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [LumiUI](../../Packages/LumiUI) | UI components |
