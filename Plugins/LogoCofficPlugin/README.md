# LogoCofficPlugin

Lumi plugin: Coffic logo display.

## Features

- **Logo Display** — Shows Coffic logo in the UI

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [LumiUI](../../Packages/LumiUI) | UI components |
