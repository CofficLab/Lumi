# DocxReadPlugin

Lumi plugin: Read and parse .docx files.

## Features

- **Docx Parsing** — Parse Microsoft Word documents
- **Content Extraction** — Extract text content from docx files

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
