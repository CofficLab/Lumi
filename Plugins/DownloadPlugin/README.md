# DownloadPlugin

Lumi plugin: Download file management.

## Features

- **File Download** — Download files from URLs
- **Progress Tracking** — Track download progress

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
