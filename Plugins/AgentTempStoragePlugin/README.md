# AgentTempStoragePlugin

Lumi plugin: Agent temporary storage management.

## Features

- **Temp Storage** — Manages temporary data for agents
- **Cleanup** — Automatic cleanup of temporary files

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
