# EditorYAMLPlugin

Lumi plugin: YAML language support for the editor.

## Features

- **Syntax Highlighting** — YAML syntax highlighting
- **Language Support** — YAML language integration

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [EditorService](../../Packages/EditorService) | Editor service |
