# EditorJSONPlugin

Lumi plugin: JSON language support for the editor.

## Features

- **Syntax Highlighting** — JSON syntax highlighting
- **Language Support** — JSON language integration

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [EditorService](../../Packages/EditorService) | Editor service |
