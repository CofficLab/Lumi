# EditorSwiftPlugin

Lumi plugin: Swift language support for the editor.

## Features

- **Syntax Highlighting** — Swift syntax highlighting
- **Language Support** — Swift language integration
- **Tree-sitter** — Using tree-sitter for parsing

## Requirements

- macOS 14.0+
- Swift 6.0+

## Dependencies

| Package | Description |
|---------|-------------|
| [LumiCoreKit](../../Packages/LumiCoreKit) | Core framework for Lumi plugins |
| [EditorService](../../Packages/EditorService) | Editor service |
