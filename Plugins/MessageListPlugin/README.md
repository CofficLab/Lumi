# 💬 ChatMessagesPlugin (AgentChat)

Agent 聊天插件，负责右侧栏的消息列表 Section，显示当前会话的聊天消息时间线。

## 功能

- **消息时间线** — 在右侧栏展示当前会话的完整聊天消息
- **聊天气泡** — 区分用户与助手消息的气泡渲染
- **Token 进度条** — 展示 token 使用量进度
- **延迟进度条** — 展示响应延迟信息
- **原始消息切换** — 可展开查看消息原始内容
- **空状态视图** — 无消息时的友好提示

## UI 贡献

| 方法 | 说明 |
|------|------|
| `addSidebarSections` | 右侧栏消息列表 Section（仅在 AI Chat 上下文中显示） |

## Policy

`.alwaysOn` — 核心聊天 UI 插件，不允许用户禁用。
